Internal

Copilot sub-agents

Google-Search & URL-fetch specialists — ground terminology in Fabric docs. Packaged inside the Copilot.

MCP proxies (Cloud Run)

Elastic proxy (adds save_rule / save_results write tools) · Fabric REST proxy. Keep credentials out of agent code.

Vertex Agent Engine

Each main actor is a Reasoning Engine deploy, called over :query. SDK-deployed to dodge the Studio MCP bug.

One responsibility per agent keeps the audit trail legible — a failure is attributable to one agent and one tool call, not buried in a monolith.
